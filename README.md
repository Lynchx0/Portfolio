Welcome to the repository for my personal portfolio website. This project is a visual representation of my skills as a **Cybersecurity Analyst** and my hands-on experience in SOC operations and incident response.

## 🚀 Project Overview
This is a fully animated, single-page website built using pure HTML, CSS, and vanilla JavaScript. It is designed to showcase technical expertise through a high-end, interactive interface reflecting the nature of the cybersecurity field.

### 🎨 Key Features:
*   **Live Particle Canvas:** A dynamic background animation representing network connectivity.
*   **3D Interactive Elements:** Floating cards and pulsing elements to highlight key career stats.
*   **Performance Focused:** Built without external frameworks or libraries for maximum speed and clean code.
*   **Fully Responsive:** Optimized for all devices, from mobile screens to large desktop monitors.
