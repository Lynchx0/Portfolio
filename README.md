# 🛡️ Abdalla Nofal | SOC Analyst Portfolio

موقع شخصي (Portfolio) تفاعلي يعرض السيرة الذاتية، المهارات التقنية، والخبرات العملية في مجال الأمن السيبراني وتحليل العمليات الأمنية (SOC). يتميز التصميم بواجهة عصرية، أنظمة حركة متقدمة، وأداء عالٍ باستخدام لغات الويب الأساسية فقط.

## 🚀 معاينة مباشرة (Live Demo)
يمكنك فتح ملف `index.html` مباشرة في أي متصفح لرؤية الموقع بكامل مميزاته.

## ✨ المميزات (Features)
* **تصميم مستقبلي:** خلفية سوداء عميقة مع تأثير "Particle Canvas" (نقاط وخطوط متصلة تتحرك وتتفاعل).
* **تجربة مستخدم فريدة (UX):** مؤشر ماوس مخصص (Custom Cursor) مع حلقة تتبع مضيئة.
* **أنظمة حركة متقدمة:** * شاشة تحميل (Loading Screen) احترافية تظهر الأحرف الأولى من الاسم.
    * تأثير "3D floating" للبطاقة التعريفية في القسم الأول.
    * تأثير "Scroll Reveal" لظهور العناصر تدريجياً أثناء التمرير.
* **عرض المهارات:** أشرطة مهارات متحركة تعمل بمجرد الوصول إليها (Intersection Observer) مع "Shimmer sweep".
* **استجابة كاملة (Responsive):** الموقع متوافق تماماً مع الهواتف الذكية، الأجهزة اللوحية، وأجهزة الحاسوب.

## 🛠️ التقنيات المستخدمة (Tech Stack)
المشروع مبني بالكامل بدون استخدام أي مكتبات خارجية أو Frameworks لضمان السرعة والبساطة:
* **HTML5:** لهيكلة المحتوى بشكل دلالي (Semantic HTML).
* **CSS3:** للتنسيق، الأنيميشن (Keyframes)، وتأثيرات الزجاج (Frosted Glass).
* **Vanilla JavaScript:** للتحكم في حركة الـ Particles، الـ Custom Cursor، ونظام الـ Scroll Reveal.
* **Google Fonts:** استخدام خطوط "Syne" للعناوين و "DM Sans" للنصوص.

## 📁 محتوى الموقع
1.  **القسم الرئيسي (Hero):** يضم الحالة العسكرية، المسمى الوظيفي، وبطاقة تعريفية ثلاثية الأبعاد.
2.  **قسم المهارات (Skills):** عرض لأدوات الأمن السيبراني (SIEM, Firewalls, Penetration Testing) مع Proficiency Bars.
3.  **الخبرة العملية (Experience):** خط زمني (Timeline) تفاعلي يوضح المسار المهني من Al Ahly Leasing إلى CyberTalents و Red Nexus.
4.  **المشاريع (Projects):** شبكة تعرض أهم المشاريع مثل "College Network Infrastructure" و "SOC Framework Architecture".
5.  **التعليم واللغات:** عرض الشهادات الأكاديمية واللغات (عربي، إنجليزي، ألماني).
6.  **التواصل (Contact):** قسم تواصل سريع مع تأثيرات Ripple جذابة.

## 💻 كيفية التشغيل
1. قم بتحميل ملف الـ HTML.
2. انقر مرتين على الملف لفتحه في أي متصفح (Chrome, Firefox, Edge).
3. لا يحتاج المشروع لأي عملية تثبيت أو Server
